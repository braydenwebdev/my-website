Web Design

Learn to design and build modern, accessible, and responsive websites using industry‑standard tools and front‑end development practices. This program blends visual design, UX principles, and hands‑on coding to prepare students for real‑world web projects.

What You’ll Learn

Core skills developed in this program.

Design Skills

  • Responsive layout & grid systems
  • UI design fundamentals
  • Color, spacing, and typography for the web
  • Wireframing & prototyping

Technical Skills

  • HTML, CSS, and modern JavaScript
  • Accessibility (A11Y) best practices
  • Web performance & optimization
  • Version control with Git

Career Paths

Graduates often pursue roles such as:

  • Web Designer
  • Front‑End Developer
  • UI/UX Designer
  • Interactive Media Designer
